Food Ordering App
 Food Ordering App

A responsive food ordering web application built using the MERN (MongoDB, Express, React, Node.js) stack. This app allows users to browse the menu, place orders, and manage their cart. The admin panel provides functionality to manage food items, view orders, and track user activity.

🚀 Demo
Check out the live demo: Food Ordering App

✨ Features
User Interface:

🧑‍🍳 Browse the menu with categories
🛒 Add and remove items from the cart
🔒 User authentication (login and signup)
📦 Order summary and checkout process
📱 Responsive design for mobile and desktop
Admin Panel:

🍲 Manage food items (add, update, delete)
📝 View all orders placed by users
⏱️ Real-time order status tracking
🛠️ Tech Stack
Frontend:

 React.js
 Redux Toolkit
 Tailwind CSS
 React Router
Backend:

 Node.js
 Express.js
 MongoDB (with Mongoose for data modeling)
 JWT for authentication
Deployment:

 Frontend: Render.com
 Database: MongoDB Atlas
